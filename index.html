Cara Upload

  1. Pilih file raport di bawah untuk membuat pratinjau lokal (tidak tersimpan di server).
  2. Setelah memilih file, klik tombol Buka Formulir untuk membuka Google Form dan unggah file di sana.
  3. Untuk melihat hasil upload, klik Lihat Hasil menuju Google Sheets.

File tidak dikirim otomatis — Anda harus mengunggahnya lewat Google Form.
